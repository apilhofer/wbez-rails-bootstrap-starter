# WBEZ Bootstrap Design Rules

## Brand character
- Editorial, restrained, high-contrast, trustworthy
- Typography-led rather than decoration-led
- Neutral surfaces with red used for emphasis and primary actions

## Typography
Sans UI font:
"Source Sans Pro", Arial, sans-serif

Serif editorial font:
"Charter", "Palatino Bold", serif

## Core colors
- Text primary: #252525
- Text strong: #000000
- Text muted: #636466
- Brand primary: #ed0000
- Brand hover: #cf0000
- Border: #dcddde
- Canvas: #ffffff
- Subtle surface: #f3f3f3
- Success: #11a06b
- Error: #f51c1c
- Info: #1d9cd3
- Warning: #ffd03d

## Layout widths
- Reading width: 720px
- Content width: 1080px
- App shell: 1300px

## Usage rules
- Prefer white and off-white surfaces
- Use red for primary CTAs, active states, and emphasis
- Use serif intentionally
- Keep forms and dense app UI sans-serif
- Prefer subtle borders over heavy shadows
